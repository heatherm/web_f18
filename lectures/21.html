Custom Domain

how to get myapp.herokuapp.com to be myapp.com

NS record

Specify the servers which are providing DNS services for the domain name

A record

Address record maps domain to IP

example.com -> 1.2.3.4

CNAME record

Map a subdomain such as www or mail to the domain hosting

www.example.com -> example.herokuapp.com

MX record

Specifies a mail server responsible for accepting email messages on behalf of a domain

Example: ASPMX.L.GOOGLE.COM.

1. Buy a Domain

1a. Get a DNS Service (optional)

2. Tell Heroku

            
heroku domains:add example.com
heroku domains:add www.example.com

3. Tell your DNS

            
                CNAME www example.herokuapp.com

4. Redirect to www
